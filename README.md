## Specifications
### Operating voltages
- Transmitter unit parts:
	- HTTM capacitive touch sensor: 2,7...6V (Output voltage: +3.3 V ± 0.1 V, Current: 500mA)
	- Transmitter: FS1000A: 2,5...12V (20..28mA)
	- Micro-controller ATtiny45: 2.7 - 5.5V (sleep: 1.8V 300 μA)
- Reciever: FS1000A: 5V
